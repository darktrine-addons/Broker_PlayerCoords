# Broker: Coords

A lean LibDataBroker plugin that displays the World of Warcraft player's location.

## Features (planned for v0.1)

- Current zone, subzone, and coordinates in the broker text
- Tooltip on hover with zone, subzone, sanctuary/contested status, continent, and coordinates
- Click to toggle the world map
- Shift-click to share a waypoint hyperlink in chat
- Configurable update rate (0.5 – 5 Hz; events-driven when standing still)
- Configurable coordinate precision (0, 1, or 2 decimals)
- Standard WoW Settings panel; full localization framework (enUS first)

## Status

**Pre-alpha.** Folder scaffold only — functionality is not yet wired in.

## Compatibility

- Retail (Interface 12.0.x)
